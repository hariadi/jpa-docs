# Pemasangan #

  1. Muat turun template SBPA terbaru.
  1. Extract ke dalam root server
  1. Muat naik docs.zip melalui FTP yang disediakan
  1. Extract dan letakkan folder `/docs/` ke dalam folder `/sbpa/`
